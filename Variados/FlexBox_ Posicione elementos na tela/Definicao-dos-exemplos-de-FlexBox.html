Neste exemplo, temos um container com três itens dentro. Usamos JavaScript para acessar o contêiner e os itens usando querySelectore querySelectorAll. Em seguida, definimos as propriedades CSS necessárias no contêiner e nos itens usando o styleobjeto. Ao definir displaycomo 'flex', o contêiner se torna um contêiner flexível. A flexDirectionpropriedade controla a direção do layout ( 'row'para horizontal ou 'column'vertical). A alignItemspropriedade define como os itens se alinham verticalmente dentro do contêiner e justifyContentcontrola a distribuição horizontal dos itens. Para os itens, definimos flexBasispara definir seu tamanho inicial e flexGrowpermitir que eles cresçam e preencham qualquer espaço restante no contêiner. Este é um exemplo básico e há muito mais propriedades e opções disponíveis no flexbox. Você pode explorar todos os recursos do flexbox consultando a documentação e os tutoriais do CSS.
